# Nhánh Develop - Tích hợp code

Nhánh này dùng để tích hợp code từ các nhánh feature.

## Quy trình:
1. Mỗi thành viên tạo Pull Request từ nhánh feature của mình vào develop
2. Nhóm trưởng (Ninh) review và merge
3. Khi tất cả hoàn thành, merge develop vào master

## Trạng thái:
- [ ] Ninh - feature/student-management
- [ ] Nguyên - feature/question-management
- [ ] Phong - feature/exam-result
- [ ] Hùng - feature/admin-report
